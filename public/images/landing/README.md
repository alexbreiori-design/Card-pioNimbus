# Imagens da landing page

Coloque aqui os prints do sistema para a vitrine comercial da Nimbus.

## Estrutura sugerida

```
landing/
  hero/
    cardapio-mobile.webp
    painel-pedidos.webp
  features/
    cardapio-mobile.webp
    entrega-zonas.webp
    segmentos.webp
    marmitas.webp
    pixel-facebook.webp
    painel-pedidos.webp
  social/
    depoimento-1.webp
```

Formato recomendado: **WebP**, largura entre 1200px e 1600px.

Os caminhos já estão referenciados em `lib/landing/content.js`. Enquanto o arquivo não existir, a página exibe um placeholder automático.
