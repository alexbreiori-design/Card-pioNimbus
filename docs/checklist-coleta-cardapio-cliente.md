# Checklist de coleta de cardápio — enviar ao cliente

Copie e adapte por WhatsApp/e-mail. Peça **uma seção por vez** se o cardápio for grande.

---

Olá! Para cadastrarmos seu cardápio no Nimbus, preciso das informações abaixo. Pode responder em texto, áudio transcrito ou planilha simples — não precisa ser técnico.

**Importante:** fotos enviamos depois; agora preciso só de nomes, descrições e preços.

---

## 1. Adicionais e complementos (enviar primeiro)

Para cada grupo (ex.: Molhos, Bordas, Proteínas extras):

- [ ] Nome do grupo
- [ ] Itens do grupo (nome de cada um)
- [ ] Preço de cada item
- [ ] O cliente precisa escolher? (sim/não)
- [ ] Se sim: mínimo e máximo de escolhas
- [ ] Escolha única ou pode marcar vários?

---

## 2. Produtos (burgers, porções, bebidas, sobremesas…)

Para cada **categoria** (ex.: Burgers, Combos, Bebidas):

- [ ] Nome da categoria

Para cada **produto**:

- [ ] Nome
- [ ] Descrição (ingredientes ou texto do cardápio)
- [ ] Preço
- [ ] Código do PDV (se tiver)
- [ ] É combo? (sim/não)
- [ ] Disponível só delivery, só mesa, ou ambos? (se souber)

---

## 3. Pizzas (se aplicável)

### Tamanhos
- [ ] Quais tamanhos vendem? (ex.: Broto, Média, Grande, Família)
- [ ] Quantas fatias cada um? (se souber)

### Sabores
Para **cada sabor**:
- [ ] Nome
- [ ] Descrição / ingredientes
- [ ] Preço em **cada tamanho** (tabela completa)

### Categorias no cardápio (como o cliente vê)
- [ ] Nomes das seções (Tradicionais, Doces, Premium…)
- [ ] Quais sabores entram em cada seção
- [ ] Quais tamanhos aparecem em cada seção
- [ ] Quantos sabores por pizza? (1, 2, 3…)
- [ ] Preço quando meio a meio: do mais caro ou média?

### Extras de pizza
- [ ] Bordas recheadas? Nome e preço de cada
- [ ] Adicionais na pizza? Nome e preço

---

## 4. Marmitas (se aplicável)

### Grupos
- [ ] Como organizam? (ex.: “Almoço seg–sex”, “Marmita fit”)

### Para cada marmita / dia
- [ ] Nome que aparece pro cliente
- [ ] Dia da semana (ou “todos os dias”)
- [ ] Descrição
- [ ] Tamanhos e preço de cada (Mini, Média, Grande…)

### Montagem (passos)
Para cada etapa que o cliente escolhe:
- [ ] Título do passo (ex.: “Escolha 1 proteína”)
- [ ] De qual grupo de adicionais vêm as opções? (mesmo nome do item 1)
- [ ] É obrigatório?
- [ ] Mínimo e máximo de escolhas

### Horário
- [ ] Marmita só em horário específico? (ex.: 11h–14h)

---

## 5. Confirmações finais

- [ ] Segmento da loja: hamburgueria / pizzaria / marmita / misto?
- [ ] Algo está em promoção fixa no cardápio? (tratamos depois em Promoções)
- [ ] Algum item indisponível ou sazonal?

---

## O que você faz depois de receber

1. Cola as respostas no **agente conversor** (ver `docs/nimbus-catalog-import-agent.md`)
2. Gera o `cardapio-[slug].json`
3. Super Admin → loja → aba **Cardápio** → Validar → Importar
4. Pede ao cliente (ou sobe você) as **fotos** no admin da loja
